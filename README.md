#  🍴Mamta Bhojnalaya - Responsive Restaurant Website
# 📖About the Project
Mamta Bhojanalaya is a modern and responsive restaurant website created to provide customers with an engaging online experience. The website highlights the restaurant's specialties, menu, gallery, customer feedback, and contact information through a clean and easy-to-use interface. It is optimized for desktops, tablets, and smartphones.

# ▶️Running the Project
1.Download or clone this repository.
2.Open the project folder.
3.Verify that all images are placed inside the images folder.
4.Launch index.html in your preferred web browser.
5.For the best experience, use the Live Server extension in Visual Studio Code.
